# Sales Data Analysis (SQL + Power BI)

This project analyzes sales performance using SQL and visualizes insights using Power BI.

## Tools Used

- SQL Server
- Power BI
- GitHub

## Project Structure

Dataset  
Raw sales dataset used for analysis.

SQL  
Contains queries for data cleaning, exploration, and advanced analysis.

PowerBI  
Interactive dashboard visualizing key business insights.

## Key Insights

- West region generates the highest sales.
- Technology category has the highest profit margin.
- Top customers contribute a significant portion of total revenue.
- Sales show strong growth in recent years.
